
Olá professor Adriano, está é uma rede Neural desenvolvida em Java para processamento de números escritos à mão. 
Esse é um projeto desenvolvido para aprender mais sobre a área, os cálculos foram realizados tendo como referência as aulas
do canal do YouTube3b1b sobre aprendizagem de máquina na prática. A rede neural usa backprogragation como método de treinamento. 

Link :  https://www.youtube.com/watch?v=aircAruvnKk&list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi


Para o treinamento é usado uma base de dados do mnist disponível na internet.
